# About

simple ElectronJS App that showcases a Login form for Angoway System

# Running Locally 

1. Clone the repo
``` powershell
git clone https://github.com/DevDario/electronjs-starter-app

```

2. Access the folder 
``` powershell
cd electronjs-starter-app

```

3. Install dependencies
``` powershell
npm install
```

4. Run the app
``` powershell
npm run start
```
